# eagles-transport-dz
نحن شركة نقل بالشاحنات افتراضية محترفة (VTC) مكرسة لتوفير قوافل واقعية، وقيادة آمنة، وعمل جماعي، ومجتمع ودود للاعبي Euro Truck Simulator 2 و American Truck Simulator.  مهمتنا هي جمع السائقين معًا، وتنظيم فعاليات قوافل عالية الجودة، وخلق تجربة ممتعة لكل عضو. سواء كنت سائق شاحنة متمرسًا أو بدأت رحلتك للتو، ستجد دائمًا مكانًا لك بيننا.
